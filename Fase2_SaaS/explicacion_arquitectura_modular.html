2. Flujo de Trabajo en Segundo Plano (Asíncrono)

Por qué podés cerrar el navegador y el sistema sigue trabajando solo:

01

Carga Rápida

Subís el Excel. El sistema lo recibe y te libera el navegador en menos de 2 segundos.

02

Cola de Tareas

La tarea entra a una "lista de espera" (Redis) de alta velocidad manejada por la API.

03

Analista IA (Worker)

Un "contenedor" se enciende, toma la tarea, consulta la normativa y genera el informe.

04

Notificación

El sistema te avisa por el Dashboard (o mail) que los reportes están listos para descargar.

🚀 El Poder de la Concurrencia

A diferencia del portal actual que procesa una cosa a la vez, la fase modular permite:

  • Multi-Analista: Podés procesar 50 auditorías de 50 clientes distintos al mismo tiempo.
  • Aislamiento: Si el módulo de IVA tiene un error, el de IIBB sigue funcionando perfecto.
  • Actualización "en caliente": Podés cargar una nueva ley de CABA 2027 sin apagar el sistema.

3. Tecnologías Core & Costos Operativos

Cada pieza del rompecabezas fue elegida para ser gratuita en software y eficiente en hospedaje:

⚡ FastAPI (Motor Python)

Rol: El director de orquesta. Recibe pedidos y los reparte a los módulos.

Costo Software: US$ 0

Hospedaje: ~US$ 7-12/mes (Docker Cloud)

🐘 PostgreSQL (Base de Datos)

Rol: El archivo central. Guarda el historial de todos los clientes y módulos.

Costo Software: US$ 0

Hospedaje: ~US$ 10-15/mes (DB Gestionada)

Total Infraestructura Base: ~US$ 30 - 50 / mes para soportar todos los módulos futuros.

🌐 La Puerta de Entrada (Frontend Next.js)

La página web no necesita un servidor aparte. La incluimos dentro del mismo esquema Docker. Esto significa que heredará toda la seguridad del backend y no sumará ni un solo peso al presupuesto de hospedaje.

Tecnologías Involucradas (Stack Recomendado)
